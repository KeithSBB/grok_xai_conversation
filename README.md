# grokzilla
Home Assistant HACS custom_component for Grok LLM integration using xai-sdk-python api
and Home Assistant's LLM API.

## Goals
1. Use the xAI python SDK
1. Leverage xAI's Stateful Responses API
1. Use the Home Assistant LLM API
1. Add custom tools by subclassing llm.Tool
1. Minimize tokens and Grok server bandwidth
1. Optimize response time
